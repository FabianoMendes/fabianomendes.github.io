Santander Customer Satisfaction¶

Definição Problema de Negócio¶

A satisfao do cliente uma medida fundamental de sucesso. Clientes insatisfeitos cancelam seus servios e raramente expressam sua insatisfao antes de sair. Clientes satisfeitos, por outro lado, se tornam defensores da marca!

O Banco Santander est pedindo para ajud-los a identificar clientes insatisfeitos no incio do relacionamento. Isso permitiria que o Santander adotasse medidas proativas para melhorar a felicidade de um cliente antes que seja tarde demais.

Neste projeto de aprendizado de mquina, voc trabalhar com centenas de recursos annimos para prever se um cliente est satisfeito ou insatisfeito com sua experincia bancria.

Defina claramente o problema de negcio, faa a coleta e preparo dos dados, escolha um algoritmo, treine o modelo e avalie a acurcia, que deve ser de pelo menos 70%.

Fonte de dados¶

https://www.kaggle.com/c/santander-customer-satisfaction

train.csv - Conjunto de treinamento incluindo varivel target

test.csv - Conjunto de teste sem varivel target

sample_submission.csv - Arquivo de envio de amostra no formato correto

Função plot ROC¶

In [34]:
def plot_roc_curve(modelo, X_teste, Y_teste):
    
    prob = modelo.predict_proba(X_teste)
    prob = prob[:, 1]
    fper, tper, thresholds = roc_curve(Y_teste, prob)
    
    plt.plot(fper, tper, color='red', label='ROC')
    plt.plot([0, 1], [0, 1], color='green', linestyle='--')
    plt.xlabel('Taxa de falsos positivos')
    plt.ylabel('Taxa de verdadeiros positivos')
    plt.title('Curva ROC')
    plt.legend()
    plt.show()
    
    return fper, tper
